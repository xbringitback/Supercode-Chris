ul vs. ol

  • unordered list
    • HTML-tag: ul
    • listet Punte mit bullet point auf
    • Beispiele:
      • to-Do Liste
      • Navigationsmenü
      • Kommentare
      • Vorteile auflisten
      • Links
      • FAQ-Liste
      • Referenzen
    • Weiterführende Infos
  • ordered list
    • HTMl-tag: ol
    • listet punkte mit Nummerierunge auf
    • Beispiele:
      1. Inhaltsverzeichnis
      2. Einkaufsliste
      3. Anweiseungen
      4. Rangfolge
      5. Quiz
      6. Toplisten
    • Weiterführende Infos
